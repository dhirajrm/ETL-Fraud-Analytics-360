## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/my-new-repo.git
   ```
2. Navigate to the project directory:
   ```
   cd my-new-repo
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To run the application, use the following command:
```
npm start
```
